===============================================
TROUBLESHOOTING
===============================================

Problem: ZIP file is too large (>1 GB)
Solution: Check for and remove:
  - venv/ folder
  - node_modules/ folder
  - Large dataset files
  - Multiple model versions
  - Log files

Problem: Project doesn't run after extraction
Solution: 
  - Verify requirements.txt is included
  - Check for hardcoded paths
  - Ensure model files are present
  - Test following your own README.md

Problem: Missing files in ZIP
Solution:
  - Don't select files individually
  - Compress the entire folder
  - Verify before finalizing
